## 常见问题

### Q: 推荐使用什么 AI 工具？

A: 推荐使用以下 AI 工具：
- **VS Code + GitHub Copilot**：可以直接操作文件系统
- **Cursor**：强大的 AI 编码助手
- **ChatGPT / Claude**：可以通过对话生成代码

### Q: AI 生成的代码有问题怎么办？

A: 建议按以下步骤处理：
1. 检查 AI 是否使用了正确的文档版本
2. 向 AI 提供更详细的需求描述
3. 让 AI 分步生成，逐步验证
4. 参考示例模板，提供更具体的示例

### Q: 如何让 AI 生成更高质量的代码？

A: 建议：
1. 提供详细的需求描述，包括所有细节
2. 提供具体的示例，包括输入和输出
3. 明确指定使用的文档版本
4. 要求 AI 添加注释和文档字符串
