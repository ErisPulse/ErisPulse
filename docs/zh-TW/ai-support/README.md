## 常見問題

### Q: 推薦使用什麼 AI 工具？

A: 推薦使用以下 AI 工具：
- **VS Code + GitHub Copilot**：可以直接操作檔案系統
- **Cursor**：強大的 AI 編碼助手
- **ChatGPT / Claude**：可以透過對話產生程式碼

### Q: AI 產生的程式碼有問題怎麼辦？

A: 建議按以下步驟處理：
1. 檢查 AI 是否使用了正確的文件版本
2. 向 AI 提供更詳細的需求描述
3. 讓 AI 分步產生，逐步驗證
4. 參考範例模板，提供更具體的範例

### Q: 如何讓 AI 產生更高品質的程式碼？

A: 建議：
1. 提供詳細的需求描述，包括所有細節
2. 提供具體的範例，包括輸入和輸出
3. 明確指定使用的文件版本
4. 要求 AI 加入註解和文件字串
