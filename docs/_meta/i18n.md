# 多语言文档系统说明

## 概述

ErisPulse 文档系统已升级为支持多语言的架构，文档按语言代码组织在 `docs/` 目录下。

## 目录结构

```
docs/
├── zh-CN/          # 简体中文
│   ├── README.md
│   ├── getting-started/
│   ├── user-guide/
│   └── ...
├── en/             # 英文
│   ├── README.md
│   ├── getting-started/
│   └── ...
├── zh-TW/          # 繁体中文
│   └── ...
└── _meta/          # 元数据和索引
    ├── README.md
    ├── zh-CN/       # 中文索引
    │   ├── docs-mapping.json
    │   └── docs-search-index.json
    ├── en/         # 英文索引
    │   ├── docs-mapping.json
    │   └── docs-search-index.json
    ├── zh-TW/      # 繁体中文索引
    │   ├── docs-mapping.json
    │   └── docs-search-index.json
    ├── docs-mapping.json         # 兼容性索引（已弃用）
    └── docs-search-index.json    # 兼容性索引（已弃用）
```

## 工具使用

### 1. 索引生成器 (generate-docs-index.py)

为所有语言生成索引：
```bash
python scripts/tools/generate-docs-index.py
```

为特定语言生成索引：
```bash
python scripts/tools/generate-docs-index.py --lang en
python scripts/tools/generate-docs-index.py --lang zh-CN
python scripts/tools/generate-docs-index.py --lang zh-TW
```

**输出位置**：
- 新索引：`docs/_meta/{lang}/`
- 兼容性索引（已弃用）：`docs/_meta/`

### 2. AI 物料生成器 (generate-ai-prompts.py)

为所有语言生成 AI prompt：
```bash
python scripts/tools/generate-ai-prompts.py
```

为特定语言生成 AI prompt：
```bash
python scripts/tools/generate-ai-prompts.py --lang en
```

**输出位置**：`docs/{lang}/ai-support/prompts/`

### 3. API 文档生成器 (generate-api-docs.py)

为所有语言生成 API 文档：
```bash
python scripts/tools/generate-api-docs.py
```

为特定语言生成 API 文档：
```bash
python scripts/tools/generate-api-docs.py --lang en
```

**输出位置**：`docs/{lang}/api-reference/auto_api/`

**说明**：
- API 文档从源代码自动生成
- 默认从中文生成，然后复制到其他语言目录
- 因为 API 文档内容主要是代码，翻译意义不大

### 4. 文档翻译器 (translate-docs.py)

翻译文档到其他语言：
```bash
python scripts/tools/translate-docs.py
```

只翻译到特定语言：
```bash
python scripts/tools/translate-docs.py --lang en
```

**说明**：
- 需要配置 API 密钥（见 `translate-config.json`）
- 自动缓存已翻译的文件
- 支持增量翻译

## 兼容性说明

### 弃用标记

旧版本的索引文件（`docs/_meta/docs-mapping.json` 和 `docs/_meta/docs-search-index.json`）仍会生成，但已标记为弃用：

```json
{
  "_deprecated": true,
  "_note": "此索引已弃用，请使用 docs/_meta/{lang}/ 目录下的语言特定索引"
}
```

### 路径格式

新索引中的文档路径包含语言代码：
- 旧格式：`getting-started/README.md`
- 新格式：`zh-CN/getting-started/README.md`

## 添加新语言

1. 在 `docs/` 下创建新的语言目录（如 `ja`、`ko` 等）
2. 复制或翻译文档内容
3. 更新 `translate-config.json` 中的 `target_langs`
4. 运行相应的生成器

## 注意事项

1. **索引生成**：每次文档更新后都需要重新生成索引
2. **AI prompt**：只在源语言更新后需要重新生成
3. **API 文档**：源代码更新后需要重新生成
4. **翻译**：源文档更新后需要重新翻译或手动更新
5. **兼容性索引**：会自动生成，但建议逐步迁移到新格式
