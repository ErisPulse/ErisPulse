# AI-Assisted Development

This guide helps you use AI to quickly generate module and adapter code that complies with ErisPulse specifications.

## 📦 AI Documentation Assets

ErisPulse provides comprehensive AI development documentation assets, which can be directly fed to AI to generate code that complies with the specifications.

## Frequently Asked Questions

### Q: What AI tools are recommended?

A: The following AI tools are recommended:
- **VS Code + GitHub Copilot**: Can directly manipulate the file system
- **Cursor**: A powerful AI coding assistant
- **ChatGPT / Claude**: Can generate code through dialogue

### Q: What should I do if the code generated by AI has issues?

A: It is recommended to follow these steps:
1. Check if the AI is using the correct documentation version
2. Provide more detailed requirement descriptions to the AI
3. Ask the AI to generate step-by-step and verify progressively
4. Refer to example templates and provide more specific examples

### Q: How can I get AI to generate higher quality code?

A: Suggestions:
1. Provide detailed requirement descriptions, including all details
2. Provide specific examples, including input and output
3. Explicitly specify the documentation version to use
4. Ask the AI to add comments and docstrings
